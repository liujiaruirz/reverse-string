# Reverse String
This repository provides five different methods in C++ to reverse a string.

## Compile/Run Instructions
### Reverse a String Main Program
To Compile:
```
g++ -std=c++11 -o main main.cpp
```
To Run:
```
./main
```

### Unit Tests
To Compile:
```
g++ -std=c++11 -o test test.cpp
```
To Run:
```
./test
```
